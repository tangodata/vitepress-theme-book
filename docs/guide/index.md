# About Vitepress-Theme-Book

This theme is used for Tangodata's online books and documentations.

It is based on Vitepress 1.0.0-alpha

## Frameworks

Core technolgies:
- Node (Javascript)
- Vite
- Vue 3
- Vitepress
- Markdown-it
- Pug

Styling:
- Sass
- TailwindCSS
   - Tailwind Typography
   - Daisy UI
- Google Fonts
- Font Awesome Icons

Markdown Extensions:
- deflist
- footnotes
- container
- toc
- attr
